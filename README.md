### 📝 실행 방법
1. Git 클론 및 브랜치 변경
  ```bash
  git clone https://github.com/hdddhdd/whycarno_6043.git
  cd whycarno_6043
  git checkout Web
  cd Web
  ```

2. 개발 환경 세팅 (모듈 설치)
  ```bash
  yarn # or yarn install
  ```

3. 실행
  ```
  yarn dev
  ```
