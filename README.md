# whyCARno - Server

### 🗂️ 파일 구조
```
├── README.md
└── Server
    ├── app.py    # serve실행 파일
    ├── fonts     # 폰트
    │   └── SCDream6.otf
    ├── requirements.txt
    └── templates
        └── index.html
```

### 🚀 실행 방법
1. Git 클론 및 브랜치 변경
  ```bash
  git clone https://github.com/hdddhdd/whycarno_6043.git
  cd whycarno_6043
  git checkout Server
  cd Server
  ```

2. 개발 환경 세팅
  ```bash
  pip3 install -r requirements.txt
  ```

3. 파이어베이스 private key json파일 다운로드

4. app.py 실행
  ```bash
  python3 app.py # or flask run
  ```
