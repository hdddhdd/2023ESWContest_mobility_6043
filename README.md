### 🚀 실행 방법
1. Git 클론 및 브랜치 변경
  ```bash
  git clone https://github.com/hdddhdd/whycarno_6043.git
  cd whycarno_6043
  git checkout AI
  cd AI
  ```

2. 개발 환경 세팅
  ```bash
  pip3 install -r requirements.txt
  ```

3. detect.py 실행
  ```bash
  python3 detect.py
  ```
